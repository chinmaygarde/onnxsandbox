# ONNX Sandbox

A sandbox project for exploring ONNX Runtime and authoring a Vulkan Execution Provider.

## Building and Testing

Always use the justfile tasks to build and test the project. After making any code changes, build and test the project and fix any build or test errors before finishing.

## Code Style

Follow the [Google C++ Style Guide](https://google.github.io/styleguide/cppguide.html).
